# Real Estate Portfolio Dashboard (Tableau)

## Overview
Interactive Tableau dashboard designed to analyze a multi-property real estate portfolio.

## Key Features
- Total Rental Income KPI
- Net Income KPI
- Occupancy Rate
- Vacant Units by Property
- Leases Expiring in the Next 3 Months
- Monthly Rental & Net Income Trends
- Global Filters (City, Property Type, Property Name, Date)

## Tools Used
- Tableau Public
- Data Modeling & Calculated Fields
- Dashboard Design & KPI Structuring
- SQL
